What You’ll Learn In 10 Days

In the introduction section you’ll get a general aspect of programming such as the importance of programming, basic terminologies used in programming, etc.

Day 2 is the actual beginning of programming, here you’ll discover what is programming? How to input variables, what are integers, what are data types, how to print output, etc. So you’ll get a strong foundation in a programming language.

In this class, you’ll learn about conditional statements such as if, if else, nested if, else if ladder, and finally the switch statement with simple examples.

This is one of the most important topics of programming. Here you’ll learn FOR loop with simple examples.

After learning the loop concept we take you to the array section. In this section you’ll learn how to input a value into an array, retrieve that value from an array, searching and sorting values from an array.

On the 6th day you’ll learn everything about functions and their type with proper examples.

On day 7 you’ll learn the most powerful concept of programming which is the OOP concept. You'll learn what is class, how to create one, what is an object, how to create an object, etc.

On day 8 you will learn the important concept of OOPs with simple examples. These concepts are so useful for your programming interview.

On day9 you’ll learn some other concepts of OOPs. After day 9 you’ll gain the potential for attending a coding interview because you learned the OOPs.

On day 10 which is the final day of our class, you’ll do a real calculator project from scratch. you can add this project to your resume, this will help you to build a good portfolio.
